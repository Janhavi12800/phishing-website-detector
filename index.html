What it catches

Scammers rely on a handful of tricks. This extension knows all of them.

🎭

Brand impersonation

Fake PayPal, Amazon or bank pages hosted on look-alike addresses like paypal.com.account-verify.tk.

🔤

Typos & look-alikes

Catches gooogle, paypa1, amaozn, rnicrosoft — even Cyrillic letters disguised as English ones.

🚨

Warning banner

Dangerous pages get a clear red banner with plain-language reasons — before you enter a password or card number.

🧲

Bait words & cheap domains

"verify-account", "free-prize", "refund" hostnames on throwaway domain endings like .tk and .icu raise the score.

🔒

Page red flags

Password forms on unencrypted pages, forms that submit your data to a different website, deceptive links, hidden frames.

🛡️

100% private

Everything runs inside your browser. Zero network requests, zero data collection, no account needed.

How it works

1

Install the extension

2

Install in 2 minutes

1

Download the extension

⬇ Download (.zip, ~30 KB)

2

Unzip the file

Right-click the downloaded file → Extract All → remember the folder.

3

Open Chrome extensions

Type chrome://extensions in the address bar and turn on Developer mode (top-right toggle).

Load it

Click Load unpacked and select the extracted folder. The shield appears next to your address bar — done!
